# Tema-Data-Access

Creaţi o aplicaţie JavaScript care va efectua parsarea datelor JSON. Este vorba de date care sunt întrebări dintr-un quiz. Fişierul JSON cu date se poate descărca de la următorul link:

quiz.json

Citiţi conţinutul acestui fişier şi apoi parsaţi conţinutul citit în aşa fel încât să obţineţi un obiect JavaScript care va fi o reprezentare obiectuală a JSON-ului.

După parsarea JSON, obiectul obţinut trebuie folosit pentru generarea codului HTML prin care veţi prezenta întrebările quiz-ului pe pagină. Întrebările din quiz trebuie prezentate ca în forma din imaginea de mai jos:
